### 4. Confidence-Based Decision Making

**The confidence system:**

```
Confidence < 45%  → ASK
  "I need more context. Tell me more about..."

Confidence 45-75% → QUESTION
  "I have some ideas, but I need to ask: ..."

Confidence > 75%  → PROCEED
  "I'm confident. Here's what I'll do..."
```

**How it works:**
1. You ask a question or submit code
2. Meta-orchestrator calculates confidence based on knowledge base coverage
3. Confidence = (matching pieces / total pieces) × 2
4. Decision is made based on confidence level
5. Decision is logged for learning

**Example:**
```
Query: "I want to build a consciousness calculator"
Matches: 3 pieces in knowledge base
Confidence: 30%
Action: ASK
Response: "I need more context. What should the calculator compute? 
           What inputs does it take? What's the output format?"
```

## Advanced: Tuning the System

### Confidence Thresholds

```typescript
// In meta-orchestrator.ts
if (confidence < 0.45) {  // Change 0.45 to adjust
  action = 'ask';
} else if (confidence < 0.75) {  // Change 0.75 to adjust
  action = 'question';
} else {
  action = 'proceed';
}
```

- Lower thresholds = more aggressive (proceed sooner)
- Higher thresholds = more cautious (ask more often)

### Agent Matching

const score = skillMatch * agent.confidence;
